Editores de video

O Shotcut é um software bastante interessante de edição de vídeos em que a sua IU parece mais com um leitor de mídia e não com um editor. Este software torna todo o processo de edição de vídeos mais simples com vários recursos fantásticos disponíveis. Para finalidades básicas funciona perfeitamente, mas se você precisa de ferramentas mais poderosas, especialmente para gravação de vídeos profissional e avançada, esta poderá não ser a melhor opção. Contudo, com todos os recursos que tem à disposição, serve perfeitamente como um editor de vídeos decente para Linux.

O Kdenlive é um software de edição de vídeos gratuito e open-source que está disponível para GNU/Linux, FreeBSD e Mac OS X. Disponibiliza algumas excelentes funções de edição de vídeos como cortar, dividir, adicionar transições e efeitos de vídeo, copiar e colar, etc. Também suporta imensos formatos de arquivos e uma vasta quantidade de camcorders e câmeras digitais.

O Pitivi é um editor de vídeos não-linear e open-source que está disponível gratuitamente. Este software foi desenvolvido para uma edição de vídeos mais intuitiva e dispõe de funcionalidades simples como juntar, aparar, dividir e cortar clipes. O sistema de curvas do Pitivi suporta mistura de áudio e os usuários podem trabalhar com o programa a partir de teclas de atalho no teclado e "esfregões". Com este programa é possível juntar e combinar vídeos e áudio, o que é uma excelente vantagem. É o primeiro editor de vídeos open-source a suportar o formato Material Exchange (MEF). A interface de usuário amigável do Pritivi permite arrastar e largar arquivos, manipular directamente e reduz toda a complexidade do processo com uma navegação por temas nativos. Este software está disponível em vários idiomas e dispõe de um suporte de usuário manual.

O Avidemux é um software de edição de vídeos gratuito e open-source desenvolvido especialmente para tarefas simples como cortar, dividir, juntar, filtrar, codificar, etc. Funciona como um software de edição de vídeos multi-plataformas gratuito e suporta vários formatos de arquivos com codecs diferentes como AVI, MPEG, MP4, ASF, etc. Com a ajuda deste programa, você pode facilmente editar os seus vídeos e exportar os vídeos editados para o iPhone, iPad, iPod, PSP, etc.

O Blender é um software de edição gráfica em 3D que está disponível gratuitamente em open-source e que é usado principalmente para criar efeitos visuais, jogos de vídeo, filmes animados e animações interactivas em 3D. Este software está programado em Python, C e C++ e contém recursos essenciais como esfolar, modelação 3D, texturação, desmontagem UV, simulação de fumos e fluídos, entre outros efeitos. Tem também outras ferramentas especialmente dedicadas à animação gráfica como renderização, edição gráfica de vídeos, animação, esculpir, compor, corresponder movimentos, rastreamento de câmera e simulação de partículas e corpos leves. Suporta também geometria primitiva incluindo gráficos em vector 3D, e o seu sistema de rastreamento por linha de raio digital consegue exportar em diversos sistemas variados para motores de renderização internos.
